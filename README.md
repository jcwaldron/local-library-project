## Run Locally

Clone the project

```bash
  git clone https://github.com/jcwaldron/local-library-project.git
```

Go to the project directory

```bash
  cd local-library-project
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm start
```
